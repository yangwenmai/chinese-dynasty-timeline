# 鼎革千年 · 中国历代权力更迭脉络

以长卷轴形式呈现从夏商周到共和国五千年的朝代更迭，聚焦权力结构的演变、制度的兴废与历史周期的回响。

## 在线预览

直接在浏览器中打开 `index.html` 即可浏览完整内容。

此外还包含：

- **五千年朝代纵览** — 按时间比例绘制的可交互时间轴条
- **权力的游戏规则** — 10 条跨朝代的权力运行规律总结
- **参考文献与编纂者** — 二十四史及历代典籍索引（含维基百科与微信读书链接）

每个朝代章节包含：

- **开国叙事** — 王朝建立的关键路径
- **皇帝列表** — 含生卒年、在位年份、继位方式、结局标签
- **权力洞察** — 制度变迁、灭亡规律、历史启示

## 特性

- 墨卷长轴设计 — 深色墨底 + 金石点缀 + 宣纸韵味，模拟历史长卷的阅读体验
- 卷轴展开动画 — Hero 区域模拟竹简/卷轴徐徐展开
- 滚动驱动动画 — IntersectionObserver 实现章节淡入
- 视差滚动效果 — Hero 背景大字视差浮动
- 右侧悬浮导航 — 圆点+标签快速跳转各朝代（桌面端）
- 卷轴进度丝线 — 右侧金色丝线实时显示阅读进度
- 卷轴滚轴装饰 — 页面顶底的卷轴木轴视觉效果
- 响应式设计 — 适配桌面与移动端

## 技术栈

- 纯 HTML + CSS + JavaScript，单文件架构（~2900 行）
- 零依赖、零构建工具
- 字体：[Ma Shan Zheng](https://fonts.google.com/specimen/Ma+Shan+Zheng)（书法）、[ZCOOL XiaoWei](https://fonts.google.com/specimen/ZCOOL+XiaoWei)（标题）、[Noto Serif SC](https://fonts.google.com/specimen/Noto+Serif+SC)（正文）

## 本地运行

无需安装任何依赖，任选一种方式：

```bash
# 方式一：直接打开
open index.html

# 方式二：本地服务器（推荐，避免字体加载问题）
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
